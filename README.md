# arduino-challenge_MCC
# Smart Coin Bank

## 프로젝트 목표
도난 방지 기능을 가진 스마트 코인 뱅크 제작.
로드셀(Load Cell)로 동전 무게를 측정해 금액을 계산·LCD에 표시하고, 키패드로 입력한 비밀번호가 맞을 때만 서보모터가 작동해 뚜껑을 열 수 있도록 함.

---

## 주요 기능
- 로드셀(HX711) 기반 동전 무게 측정 → 동전 종류 판별 및 금액 합산 표시  
- LCD(16x2)로 현재 합계 실시간 표시  
- 4x4 키패드로 비밀번호 입력 → 비밀번호가 맞으면 서보모터로 뚜껑 개폐  
- 잘못된 비밀번호 입력 시 경고(예: LCD 메시지)  

---

## 하드웨어
- Arduino Uno  
- Load Cell + HX711 모듈  
- LCD  
- 4x4 Keypad  
- Servo Motor (SG90)    
- 점퍼선, 브레드보드 등

---

## 소프트웨어 / 라이브러리
- Arduino IDE
- Libraries:
  - HX711.h  — Load Cell 증폭기 드라이버
  - LiquidCrystal_I2C.h
  - Keypad.h — 키패드 입력
  - Servo.h — 서보 제어
    
---

## 🔎 동작 원리 (간단 설명)
1. HX711으로 로드셀에서 동전 무게(그램)를 읽음  
2. 무게 값에 대해 사전 보정을 통해 동전 종류(예: 10원, 50원, 100원, 500원)를 판별  
   - (반드시 실제로 계량하여 측정값 등록)  
3. 판별 결과로 기존 금액의 새로운 금액을 합산하여 LCD에 표시  
4. 사용자가 키패드로 비밀번호 입력  
   - 비밀번호 일치 시: 서보를 지정 각도로 회전시켜 뚜껑 열기 → 정해진 시간 후 닫음  
   - 불일치 시: LCD에 오류 표시
